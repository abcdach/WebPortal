Photon Mining

SDSU-G EE and CE Capstone Project [ Mentor: David Chkhaidze ]

Photon Mining is a project that is under development by 5 engineering students from San Diego State University â€“ Georgia. The project consists of a smart solar panel that can track the sun for maximum power efficiency. In addition to the tracking ability, the smart solar panel also uses an IoT device to send numerous data (current drawn, the voltage across the battery, the power generated, light intensity, etc.) to the server. The project will also utilize a smartphone application where the user can view all of the data in a very user-friendly environment.
