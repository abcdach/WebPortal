Vagrant

April 28, 2021 by David Chkhaidze

ამ ბლოგში განხილულია Vagrant-თან დაკავშირებული საკითხები.

მოკლე აღწერა

Vagrant-ტი წარმოადგენს (ღია კოდის) პროგრამას, რომელიც განკუთვნია იმისთვის რომ შევქმნათ, განცალკევებული ვირტულაური გარემო. ეს გარემო წარმოადგენს ოპერაციულ სისტემა ლინუქს, რომელიც აბსოლუტურად იზოლირებულია ძირითადი ოპერაციული სისტემისგან, შესაბამისად ეს გვაძლევს ჩვენ საშუალებას შევქნათ პროგრამული უზრუნველყოფა, გარე ფაქტორებიდან თავისუფალ გარემოში. Vagrant-ტი მიშაობს VirtualBox-თან ერთად.


დასაინსტალირებელი პროგრამები

Sensor studio Vagrant-ტი საჭიროებს გარკვეული პროგრამების დაინსტალირებას:

  • Vagrant Development Environment
  • VirtualBox is a powerful x86 and AMD64/Intel64 virtualization product

ინსტრუქცია (Windows)

ყველა საჭირო პროგრამების დაინსტალირების შემდეგ, შეგვიძლია დავიწყოთ ვირტულაური გარემო შექნის პროცესი. ამისთვის კომანდა cmd-ით გამოვიძახოთ Windows-სის სტანდარტული, კომანდების ინტერპრეტატორი. ამის შემდეგ ყველა შემდგომი ბრძანებები აიკრიფება ინტერპრეტატორის ტერმინალში.
პირვლე რიგში შევქმნათ ფოლდერი სადაც განთავსდება ვირტუალური გარემოსთვის საჭირო ფაილები. 

mkdir MyVAGRANT
cd MyVAGRANT
მოვახდინოთ Vagrant-ტის ინიცალიზაცია 

vagrant init hashicorp/bionic64
გავააქტიუროთ ვირტუალური მანქანა 

vagrant up
გავააქტიუროთ ვირტუალური გარემოს SSH სესია 

vagrant ssh
იმის შემდეგ რაც გაიხნება SSH სესია, ჩვენ შეგვიძლია განვაახლოთ ვირტულური გარემოს ოპერაციული სისტემა, შემდეგი ბრძანებების გამოყენებთ 

vagrant@vagrant:~$ sudo apt-get update
vagrant@vagrant:~$ sudo apt-get upgrade
SSH სესიიდან გამოსვლის შემთხვევაში აკრიფედ შემდეგი ბრძანება: 

vagrant@vagrant:~$ logout
იმისთვის რომ წავშალოთ ვირტუალური მანქანა, ამისთვის cmd ტერმინალის საშუალებით შევიდეთ ფოლდერში სადაც განთავსებულია ვირტუალური გარემო და ავკრიფო შემდეგი ბრძანება: 

vagrant destroy
იმისთვის რომ დავუბრუნდეთ ვირტუალური მანქანას, მაგალითად ოპრაციული სისტემის რესტარტის შემდეგ ამისთვის cmd ტერმინალის საშუალებით შევიდეთ ფოლდერში სადაც განთავსებულია ვირტუალური მანქანა და შემდეგ გავააქტიუროთ ვირტუალური გარემოს SSH სესია: 

vagrant up
vagrant ssh

დასკვნა

Vagrant-ტის საშუალებით ჩვენ შეგვიძლია შევქმნათ ოპერაციულ სისტემა ლინუქსზე დაფუძნებული ვირტუალური სამუშაო გარემო, რომელიც აბსოლუტურა იქნება იზოლირებული ძირიტადი ოპერაციული სისტემიდან ან სხვა ასეტივე ვირტულაური გარემოდან, რაც ძალიან მნიშვნელოვანია სხვადასხვა პროგრამული უზრუნველყოფის შექმნის და ტესტირების პროცეში.
